> Connect to Github
1. Download and install GIT > https://git-scm.com/downloads
2. git init
3. git add .
4. git remote add origin https://gitlab.com/mohamdbehery/evolvice-daily-tasks.git
5. git pull origin master --allow-unrelated-histories
6. git commit -m "first commit"
7. git push -u origin master

> Connect to Github using SSH
1. ssh-keygen -t rsa -b 4096 -C "mbehery@corelogic.com"
2. ssh-add ~/.ssh/id_rsa
3. Copy the public key (clip < ~/.ssh/id_rsa.pub)
4. Create new SSH Key in Github profile and paste this key
5. ssh-keyscan github.com >> ~/.ssh/known_hosts
